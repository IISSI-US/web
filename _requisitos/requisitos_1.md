# Introducción a los Requisitos

Los requisitos son descripciones de lo que un sistema debe hacer, los servicios que ofrece y las restricciones en su operación.

## Tipos de Requisitos

- **Funcionales**: Describen las funciones del sistema
- **No funcionales**: Restricciones sobre servicios o funciones
- **De dominio**: Derivados del dominio de aplicación

Este es un ejemplo del poder de las **colecciones de Jekyll** para organizar documentación.
